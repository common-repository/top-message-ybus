This plugin writes a message at the top of you site.

== Description ==

After instalations, you can go Top Message Ybus and set a custom message, color of the font and background color (bg color).
The message will displayed on top of all pages of you site.

== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Top Message Ybus screen to configure the plugin
4. Choose a text, color for the font, background image and save.
5. You can also use a shortcode in page or posts using [top_message message="Custom Message here"], changing text for your custom message.
6. On Widgets, you can also add the "Top Message" Widget to your sidebar or widget are and add a custom text to be displayed.


== Frequently Asked Questions ==

= Can I choose the backgground color =

Yes, you have a option to change the background color called bg color.
